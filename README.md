# kboard

QMK keymap for Lily58. 

Highly unstable, likely changing significantly.

## init

1. clone qmk firmware

```
yay -S qmk
qmk setup
```

2. add submodule

```
cd keyboards/lily58/keymaps
git submodule add https://github.com/kvoli/kboard.git kboard
cd ../../..
```

3. compile keymap using QMK

```
qmk compile -kb lily58 -km kboard
```

4. flash your board

```
qmk flash  -kb lily58  -km kboard -bl dfu
```

5. short the board on `GND` and `RST`

... with something conductive

6. done
